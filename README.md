# A Birthday to Remember

A (hopefully 🤞) heartwarming Ren'Py visual novel about siblings, friends, effort, and imperfect-but-sincere love. ❤️

## About The Game

In this short story, Felisha wakes up excited for her brother Lucas' birthday and wants to make him the perfect card. With help from her best friend Isla, she learns that handmade effort and genuine feelings matter more than perfection. She also gets to speak honestly with her brother.

You join Felisha as she:

- prepares for Lucas' celebration
- asks for your name so you can take part in the story
- asks for your help with choices
- visits Isla for crafting support
- delivers a heartfelt birthday message to Lucas

## Main Characters

- **Felisha** - Lucas' older sister, energetic and caring
- **Lucas** - the birthday boy
- **Isla** - Felisha's artistic best friend who guides the card-making process

## Core Themes

- family love and support 🫶🏻
- growth through honest communication 💓
- handmade effort over "perfect" results 💝

## Features

- linear, story-driven visual novel flow
- player name input integrated into dialogue
- expressive character sprites and scene transitions
- background music and sound effects throughout key moments
- custom Ren'Py GUI and menu screens

## Run Locally

1. Install Ren'Py from <https://www.renpy.org/>.
2. Open the Ren'Py launcher.
3. Add this project folder.
4. Launch **Happy Birthday Lucas**.

## Project Structure

```text
game/
	script.rpy      # Main story and dialogue flow
	screens.rpy     # UI screens and menus
	gui.rpy         # GUI configuration
	options.rpy     # Project and build settings
	audio/          # Music and sound effects
	images/         # Character sprites and backgrounds
```

## Troubleshooting

- If the game does not start, make sure your Ren'Py version is up to date.
- If assets are missing, confirm folders inside `game/` were copied correctly.
- If you encounter script errors, check `traceback.txt` and `errors.txt`.
